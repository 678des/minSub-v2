## インストールから起動まで

1. リポジトリをクローン

```bash
git clone [https://github.com/678des/minSub-v2.git](https://github.com/678des/minSub-v2.git)
```

2. ディレクトリに移動

```bash
cd ./my-app/
```

3. 必要なパッケージをインストール

```bash
npm install
```

4. アプリを実行

```
npm run dev
```

[http://localhost:3000](http://localhost:5173)にアクセス
